## Prettier config
This is `prettier` config for my projects. If you like this style, you can also quote it in your project.
### Usage
1. Install:
```bash
npm i @cuiqg/prettier-config -D
```
2. Configure (`package.json`):
```json
{
  "prettier": "@cuiqg/prettier-config"
}
```
